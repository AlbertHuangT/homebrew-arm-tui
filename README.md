# homebrew-arm-tui

Homebrew tap for [arm-tui](https://github.com/AlbertHuangT/Arm-TUI) — an ARM v7 assembly step debugger with a GDB-style terminal UI.

## Install

```bash
brew tap AlbertHuangT/arm-tui
brew install arm-tui
```

## Usage

```bash
arm-tui your_program.s
```

See the [main repo](https://github.com/AlbertHuangT/Arm-TUI) for full documentation.
